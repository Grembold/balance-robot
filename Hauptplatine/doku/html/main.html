Erstellen der nBot Dokumentation

Zur Erstellung der Dokumentation wird das Tool Doxygen verwendet. Um die Dokumentation selbst zu aktualisieren:
  • man lädt sich die aktuelle Doxygen Version von http://www.stack.nl/~dimitri/doxygen/ herunter.
  • Installiert Doxygen
  • Im Verzeichnis / startet man die Batchdatei make_doc.bat, um die Dokumentation zu aktualisieren. Eventuell muss man vorher den Pfad auf die Datei Doxygen.exe aktualisieren.
Die erzeugte HTML Dokumentation befindet sich im Ordner /doku
